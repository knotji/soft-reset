# Soft Reset 🌿

> รีเซ็ตใจเบา ๆ — Thai healing card generator powered by Gemini AI

A mobile-first web app where users choose a mood or write a short feeling, then receive a soft Thai healing card generated by Gemini AI.

## Setup

### 1. Clone & install

```bash
cd C:\Project\SoftReset
npm install
```

### 2. Configure environment

```bash
cp .env.example .env.local
```

Edit `.env.local` and add your Gemini API key:

```
GEMINI_API_KEY=your_gemini_api_key_here
```

Get a free API key at [Google AI Studio](https://aistudio.google.com/app/apikey).

### 3. Run locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js 15** — App Router
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** — page transitions and micro-animations
- **@google/genai** — Gemini 2.5 Flash API (server-side only)
- **html-to-image** — save card as PNG

## Features

- 6 mood options in Thai
- Optional feeling text input (up to 200 characters)
- Animated loading state with rotating messages
- Healing card with: title, healing message, tiny action, tonight reminder, story, hashtags
- Copy card text to clipboard
- Save card as PNG image
- Fallback response if Gemini API is unavailable

## Project Structure

```
app/
  page.tsx                    # Main page with step state machine
  layout.tsx                  # Root layout with Sarabun font
  globals.css                 # Global styles + Tailwind theme
  api/
    generate-reset/
      route.ts                # POST /api/generate-reset

  Hero.tsx                    # Landing hero section
  MoodSelector.tsx            # Mood chip grid
  FeelingInput.tsx            # Optional text textarea
  LoadingState.tsx            # Animated loading with rotating messages
  ResetCard.tsx               # Result healing card
  CopyButton.tsx              # Copy text to clipboard
  SaveCardButton.tsx          # Save card as PNG
lib/
  gemini.ts                   # Gemini API integration + fallback
types/
  reset.ts                    # TypeScript types and mood constants
```
